Sobre o Sistema

Este sistema foi desenvolvido para gerenciar o catálogo de vinhos da Vinharia Agnello. Por meio de JavaScript puro, é possível listar todos os vinhos do acervo, identificar vinhos com estoque baixo, calcular o estoque total da vinícola e exibir os nomes formatados em caixa alta — tudo via console.log e alert().

Nosso Acervo

Abra o Console do navegador (F12 → aba "Console") para visualizar as informações do sistema.
